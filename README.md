# LegalGPT
AI-powered Indian Legal Assistant

## Project Structure
- backend/ — FastAPI + RAG engine
- frontend/ — React + MUI interface
- streamlit_version/ — Streamlit prototype
  
## Quick Start
1. cd backend && pip install -r requirements.txt
2. cd frontend && npm install
3. Start backend: python -m uvicorn app.main:app --reload
4. Start frontend: npm start
